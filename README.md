# rackt-cli

Command line utility for automating common tasks when building React.js components.

**Note that this is still a WIP**

## Installing

```bash
$ npm install -g rackt-cli
```

## Tasks

```bash
# Initializing
$ rackt init my-component

# Building
$ rackt build

# Dev mode
$ rackt dev

# Releasing
$ rackt release

# Testing
$ rackt test

# Publish gh-pages
$ rackt pages
```
