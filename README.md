# runt

Task runner for building react components.

Designed to be used in conjunction with [originate-react-component](https://github.com/knomedia/originate-react-component).

## Installing

```bash
$ npm install -g runt
```

## Tasks

```bash
# Building
$ runt build

# Dev mode
$ runt dev

# Releasing
$ runt release

# Testing
$ runt test

# Publish gh-pages
$ runt pages
```
